# 🤖 Lucchese Bot

Многофункциональный Discord-бот для автоматизации дней рождений, управления контрактами (ролевыми наборами), отправки новостей и планирования сообщений.  
Разработан с учётом потребностей игровых сообществ и не только.

---

## ✨ Особенности

- 🎂 **Дни рождения**  
  – Добавление/удаление дат рождения пользователей.  
  – Автоматические напоминания за настраиваемое количество дней (например, 7, 1, 0).  
  – Интерактивная панель со списком и сортировкой (по ближайшему или календарю).  
  – Автоудаление записей при выходе участника с сервера.

- 📋 **Контракты (ролевые наборы)**  
  – Создание правил контракта в отдельной панели.  
  – Отдельные каналы для заявок и объявлений набора.  
  – Роли ответственного (может управлять) и участников (могут записываться).  
  – Запись/отмена записи через кнопки, управление участниками (удаление ответственным).  
  – Автоматическое удаление устаревших сообщений (через 24 часа).  
  – Отдельный канал для логов действий с контрактами.

- 📢 **Новости**  
  – Отправка новостей в заданный канал с поддержкой изображений и упоминания роли.  
  – Команда `/news` для доверенных пользователей.

- ⏰ **Планировщик сообщений**  
  – Команда `/say` с возможностью указать время отправки (ДД.ММ.ГГГГ ЧЧ:ММ МСК).  
  – Поддержка упоминаний пользователей и ролей, прикрепление изображений.  
  – Просмотр и отмена запланированных сообщений.

- ⚙️ **Управление ботом**  
  – Панель настроек с кнопками для установки каналов логов, новостей, дней напоминаний, ролей контрактов и т.д.  
  – Логирование действий в отдельный канал (уровни INFO, SUCCESS, WARNING, ERROR).  
  – Гибкая система прав: создатель (полный доступ), доверенные пользователи (ограниченный доступ).

- 🔒 **Конфиденциальность**  
  – Все данные хранятся локально в JSON-файлах.  
  – Команды `/terms` и `/privacy` для отображения условий использования.

## 📦 Требования

- Python 3.8 или выше
- Discord-бот токен (создаётся на [Discord Developer Portal](https://discord.com/developers/applications))
- Установленные зависимости из `requirements.txt`

## 🚀 Установка и запуск

1. **Клонируйте репозиторий**  
   ```bash
   git clone https://github.com/yourname/lucchese-bot.git
   cd lucchese-bot
   ```

2. **Создайте виртуальное окружение (рекомендуется)**  
   ```bash
   python -m venv venv
   source venv/bin/activate  # Linux/Mac
   venv\Scripts\activate     # Windows
   ```

3. **Установите зависимости**  
   ```bash
   pip install -r requirements.txt
   ```

4. **Настройте переменные окружения**  
   Скопируйте файл `.env.example` в `.env` и отредактируйте его (см. раздел **Настройка .env**).

5. **Запустите бота**  
   ```bash
   python rehab_10.03.2026.py
   ```

## 📁 Структура файлов данных

Все данные хранятся в текущей директории в формате JSON:

- `birthdays.json` – дни рождения пользователей.
- `panel_state.json` – состояние панели дней рождений (ID канала/сообщения).
- `reminder_config.json` – дни для напоминаний (например, `[7,1]`).
- `reminder_messages.json` – история отправленных напоминаний для автоудаления.
- `log_config.json` – ID канала логов.
- `news_config.json` – ID канала новостей.
- `scheduled_messages.json` – запланированные сообщения.
- `temp_images/` – временное хранилище изображений для отложенных сообщений.

**Контракты:**
- `contract_rules.json` – текст правил контрактов и информация о последнем редакторе.
- `contract_panel_state.json` – состояние панели правил.
- `contract_config.json` – настройки контрактов (роли, каналы, мин. участников).
- `contract_session.json` – текущая сессия набора (активность, список участников, ID сообщения).
- `contract_messages.json` – сообщения контрактов для автоудаления.
- `contract_log_config.json` – ID канала логов контрактов.
- `contract_manage_panel_state.json` – состояние панели управления набором.

**Прочее:**
- `settings_panel_state.json` – состояние панели настроек.
- `birthday_sort_mode.json` – текущий режим сортировки дней рождений (`next`/`calendar`).

## 🤖 Команды бота

### Дни рождения
| Команда | Описание | Доступ |
|--------|----------|--------|
| `/create_panel` | Создать интерактивную панель с днями рождения | только создатель |
| `/add_birthday <user> <date>` | Добавить день рождения пользователю (ДД.ММ) | доверенные |
| `/remove_birthday <user>` | Удалить запись | доверенные |
| `/refresh_panel` | Обновить панель | доверенные |
| `/restore_panel` | Восстановить панель после удаления | только создатель |

### Контракты
| Команда | Описание | Доступ |
|--------|----------|--------|
| `/create_contract_panel` | Создать панель с правилами контрактов | только создатель |
| `/create_contract_manage_panel` | Создать панель управления набором | только создатель |
| (в панели) **Открыть набор** | Создаёт сообщение для записи в канале набора | ответственный |
| (в панели) **Закрыть набор** | Закрывает набор (проверяет минимум участников) | ответственный |
| (в панели) **Заявка в Контракт** | Отправляет заявку в канал заявок | участник |
| (в панели) **Управление участниками** | Позволяет ответственному удалить участника | ответственный |
| (в панели) **Удалить набор** | Принудительно удаляет набор (без проверки) | только создатель |
| (в сообщении набора) **Записаться** | Запись в контракт | участник |
| (в сообщении заявки) **➕ Добавить в набор / Открыть набор** | Принимает заявку и добавляет пользователя | ответственный |

### Новости
| Команда | Описание | Доступ |
|--------|----------|--------|
| `/set_news_channel <channel>` | Установить канал для новостей | только создатель |
| `/unset_news_channel` | Отключить канал новостей | только создатель |
| `/news <title> <content> [image] [role]` | Отправить новость | доверенные |

### Планировщик сообщений
| Команда | Описание | Доступ |
|--------|----------|--------|
| `/say <channel> <title> <message> [users] [roles] [image] [schedule]` | Отправить сообщение сейчас или по расписанию | доверенные |
| `/list_scheduled` | Показать все запланированные сообщения | доверенные |
| `/cancel_scheduled <id>` | Отменить запланированное сообщение | доверенные |

### Управление ботом
| Команда | Описание | Доступ |
|--------|----------|--------|
| `/create_settings_panel` | Создать панель управления настройками | только создатель |
| `/set_log_channel <channel>` | Установить канал логов | только создатель |
| `/unset_log_channel` | Отключить логирование | только создатель |
| `/set_contract_log_channel <channel>` | Установить канал логов контрактов | только создатель |
| `/unset_contract_log_channel` | Отключить логи контрактов | только создатель |
| `/set_reminder_days <days>` | Установить дни напоминаний (через запятую) | только создатель |
| `/show_reminder_days` | Показать текущие дни напоминаний | все |
| `/list_trusted` | Показать список доверенных пользователей | только создатель |

### Информация
| Команда | Описание | Доступ |
|--------|----------|--------|
| `/terms` | Условия использования | все |
| `/privacy` | Политика конфиденциальности | все |
